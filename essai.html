Vision statement

Jet lag is a common problem for many travelers and for many workers. This AmI can help the user getting used to the new circadian rhythm.

The systeme will be used in hotel's room. It will help people that come from a far away country for profesional reasons or not. the user will just have to answer some questions to help the system to calculate how many time and how the user needs to sleep. The user will receive some advice to help him with the new circadian rhythm.

The problem is that people that often travel are tired because they do not sleep at the right time because they used to sleep at a precise time in their country. Here we propose a solution to permit to the user to sleep at a precise time. Like this, it will be easier for him to get used to the new circadian rhythme.

AMI main steps

Ami step Description
Sensing The ami system is capable to use the data in the smartphone (google agenda,...) to know what are the obligations of the user. It will also give advice to the user by using a telegram bot, notifications or microphone (not decided yet). Some sensors will be used to close the blinds or switch off/on the lights.
Reasoning The Ami system can calculate the circadian rythm of the user when it knows some information about his sleeping rythm.It can also understand the user's appointment and give advices to the user in using the data collected in the smartphone
Acting The Ami system is capable to switch on the lights, Close the blinds, give advice to the user with a microphone, a telegram bot or a notification
Interacting The system can give advices to the user for his schedule, suggest some naps to the user during the day,ask feedback to the user

AMI features

Ami step Description
Sensitive The Ami system can know if the user is inside the hotel room or not. If he is inside, it can collect data from his smartphone to advice him.
Reasponsive If the system calculate that the user needs to sleep, it will advice him to take a brake. The user can refuse but if he acceptes, the system will calculate the time that he needs to sleeps and fix the alarm. It will also close the blinds and switch off the lights
Adaptive The system will ask to the user some information about his sleep to calculate the best way how much he needs to sleep and when
